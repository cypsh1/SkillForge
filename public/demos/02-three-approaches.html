🪪 身份层 — "我是谁"

name: tech-news-digest-cn · version: 3.16.0
生成科技新闻摘要，具有统一的数据源模型、质量评分和多格式输出。

📦 依赖层 — "我需要什么"

系统依赖：python3(必需), mail msmtp gog gh openssl weasyprint(可选)
环境变量：11 个 API 密钥 — Twitter/X (3), Web Search (3), GitHub (4), Brave (1)

⚡ 能力层 — "我能做什么"

工具：python3 运行脚本, mail 发送邮件, gog Gmail 备用
文件：读取 config/defaults/, references/, scripts/ · 写入 /tmp/td-*, archive/

⚙️ 配置层 — "怎么定制我"

sources.json — 151 个数据源 (RSS 62 + Twitter 48 + GitHub 28 + Reddit 13)
topics.json — 主题定义、搜索关键词、显示设置
schema.json — 结构验证

▶️ 执行层 — "具体怎么跑"

run-pipeline.py → 并行运行 6 个获取器 → merge-sources.py 合并去重评分
→ generate-pdf.py / sanitize-html.py 输出 → 邮件/Discord/PDF 传送

🔧 运维层 — "出问题怎么办"

validate-config.py 配置检查 · source-health.py 源健康监控
所有脚本支持 --verbose · 自动重试 · 指数退避 · 90 天存档清理

⚡ 工具 (tools)

python3 运行数据收集和合并脚本 📦 requires.bins 📄 run-pipeline.py 📄 fetch-rss.py 📄 fetch-twitter.py 📄 +8 more
mail 邮件传送（首选） 📦 optionalBins 📄 sanitize-html.py
gog Gmail CLI 备用 📦 optionalBins
