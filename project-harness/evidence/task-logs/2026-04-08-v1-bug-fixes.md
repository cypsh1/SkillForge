### Bug #10：工具区块行高偏大
- 操作：`.tc` padding 从 `6px 10px` → `4px 8px`，margin 从 `4px` → `3px`

### Bug #9：环境变量输入框宽度定死
- 操作：添加 CSS 规则 `.et td .fi, .et td .fta { width: 100%; box-sizing: border-box }`
- 结果：输入框自适应填满单元格，各宽度下不溢出
- 文件：`src/index.css`

### Bug #5：Trigger 展示/编辑内容不一致

- 结果：展示态与编辑态字段覆盖范围完全一致
- 文件：`src/components/workspace/editor-panel.tsx`

### Bug #14：底部状态栏遮挡编辑/预览区
- 操作：
  - ContextBar 添加 Tailwind `shrink-0` 防止 flex 压缩
  - RelationBar CSS 添加 `flex-shrink: 0`
  - 编辑面板、预览面板的滚动内容区域添加 `paddingBottom: 32` px
- 结果：编辑区和预览区滚动到底时，内容完全可见，不被状态栏遮挡
- 文件：`src/components/workspace/context-bar.tsx`、`src/index.css`、`src/components/workspace/editor-panel.tsx`、`src/components/workspace/inspector-panel.tsx`
