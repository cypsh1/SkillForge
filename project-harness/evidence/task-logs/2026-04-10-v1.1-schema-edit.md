# V1.1-SCHEMA-EDIT: schema.json 可编辑

**日期**：2026-04-10
**状态**：✅ 完成

## 背景

用户排查发现 `tech-news-digest` 下的 `config/schema.json` 节点不可编辑，是 config-file 类型 JSON 中唯一的只读节点。

## 调研记录

- 排查 `editor-panel.tsx` 的 `ConfigFileEditor` 路由逻辑
- `configEditorKind()` 识别三种 config 文件：sources → SourcesEditor ✅、topics → TopicsEditor ✅、schema → SchemaViewer ❌（只读）
- `extra-file` 类型 JSON 走 `JsonFileEditor`（扁平 key-value），不适用于 schema.json 的深层嵌套结构
- 结论：需要一个原始 JSON 文本编辑器（textarea）

## 改动

| 文件 | 改动 |

| `src/components/workspace/editor-panel.tsx` | 新增 `SchemaRawEditor` 组件；`ConfigFileEditor` schema case 改用它 |
