# SkillForge

OpenClaw Skill 可视化配置工具。

## 项目状态
🚧 **开发中** — Phase 1：技术选型 + 项目骨架

## 这是什么
SkillForge 帮助 OpenClaw 用户通过可视化界面了解、配置和管理 Skill，取代手工编辑 SKILL.md 和 JSON 配置文件。

## 目标功能（MVP）
- [ ] Skill 解析与结构展示
- [ ] 可视化配置编辑
- [ ] 配置实时预览
- [ ] Skill 验证与问题检测
